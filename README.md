# News Crawl - 다리마티 뉴스 수집 & 인사이트 분석

다리마티 관련 뉴스 기사를 수집하고 인사이트를 분석하는 프로젝트

## 목표

- 다리마티 관련 뉴스 기사 자동 수집 (크롤링)
- 수집된 기사 데이터 정리 및 저장
- 키워드/트렌드 분석
- 인사이트 리포트 생성

## 프로젝트 구조

```
news-crawl/
├── README.md
├── crawlers/        # 뉴스 크롤러 스크립트
├── data/            # 수집된 기사 데이터
├── analysis/        # 분석 결과
└── reports/         # 인사이트 리포트
```

## 기술 스택

- Python
- BeautifulSoup / Requests (크롤링)
- Pandas (데이터 처리)

## 시작하기

```bash
pip install -r requirements.txt
```
